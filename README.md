# Erooster

A mail suite written in rust meant to be easy to use.

## Getting started

## Features

- Imap4rev2 compatible
- Maildir support
- TLS by default
- Single binary
- Low Resource usage
- Postgres first

## Non Goal

- Implementing every single peace of optional spec
- MySQL/Mariadb support
- Support for imap LOGIN command (It is per rev2 spec)
